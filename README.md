# GreenTrack

GreenTrack ist eine Single Page Application (SPA), die Nutzern hilft, nachhaltige Aktivitäten im Alltag zu verfolgen und zu verbessern.

## Features

- Aktivitäten anzeigen (Liste + Detail)
- Neue Einträge erstellen
- Einträge bearbeiten und löschen
- Persönliche Auswertung (z. B. CO₂-Ersparnis)
- Punktesystem zur Motivation
- Responsive Design und Fehlerbehandlung

## Technologien

- frontend: react / vue  
- styling: tailwind css / bootstrap  
- backend (optional): node.js + express  
- datenbank: mongodb / mysql
